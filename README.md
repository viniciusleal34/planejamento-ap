Documentação completa do projeto de interiores para apartamento compacto de **43,94m²**, Tipologia A, com 2 dormitórios e varanda. O projeto segue um estilo **moderno contemporâneo, clean e sofisticado**, com integração de sala, cozinha e lavanderia.

---

## 📋 Sobre o Projeto

| Atributo | Detalhe |
|---|---|
| **Área total** | 43,94 m² |
| **Tipologia** | A — 2 dormitórios + varanda |
| **Ambientes** | Sala integrada, Cozinha, Lavanderia, 2 Dormitórios, Banheiro, Varanda |
| **Estilo** | Moderno contemporâneo, clean e sofisticado |
| **Paleta principal** | Branco, cinza claro, preto e madeira clara |

### 🪑 Itens Fixos do Projeto
- **Sofá retrátil preto 180cm** — elemento principal da sala de estar
- **Geladeira inverse branca** — cozinha integrada
- **Máquina lava e seca branca Samsung** — lavanderia integrada

---

## 🎯 Objetivos do Projeto

1. Maximizar a funcionalidade e conforto em um espaço compacto de 43,94m²
2. Criar uma estética moderna, clean e sofisticada com materiais de custo-benefício
3. Integrar visualmente sala, cozinha e lavanderia por meio de piso contínuo e paleta coesa
4. Garantir boa circulação e aproveitamento de cada centímetro disponível
5. Entregar um projeto realista e executável no mercado brasileiro de 2025/2026
