## 🎯 Estratégia para Caber em R$ 40.000

Para o **cenário intermediário ficar dentro de R$ 40.000**, priorize da seguinte forma:

| Prioridade | Categoria | Nível Recomendado | Justificativa |
|-----------|-----------|------------------|---------------|
| ⭐⭐⭐ ALTA | Piso porcelanato | Intermediário | Impacto visual enorme, cobre todo o ap |
| ⭐⭐⭐ ALTA | Ar-condicionado (2 splits) | Intermediário inverter | Conforto essencial |
| ⭐⭐⭐ ALTA | Marcenaria cozinha | Intermediário | Uso diário, qualidade importa |
| ⭐⭐ MÉDIA | Guarda-roupas (2 quartos) | Econômico/Médio | Modulados bons resolvem bem |
| ⭐⭐ MÉDIA | Mão de obra | Intermediário | Não economize em execução |
| ⭐⭐ MÉDIA | Banheiro | Intermediário | Vaso suspenso + metais pretos |
| ⭐ BAIXA | Iluminação | Econômico/Médio | Spots + fita LED são suficientes |
| ⭐ BAIXA | Móveis (camas) | Econômico | Camas simples com colchão de qualidade |
| ⭐ BAIXA | Decoração | Econômico | Tapete + espelhos + cortinas básicos |

| Ação | Economia Estimada |
