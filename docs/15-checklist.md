# ✅ 15 — Checklist do Projeto

---

## 🔴 Fase 1 — Planejamento

- [ ] Definir orçamento total disponível
- [ ] Visitar lojas de referência (Leroy Merlin, MadeiraMadeira)
- [ ] Cotar gesseiro
- [ ] Cotar eletricista
- [ ] Cotar piso (porcelanato ou vinílico)
- [ ] Cotar marcenaria (armários cozinha + guarda-roupas)
- [ ] Cotar vidraçaria (box + espelhos)
- [ ] Definir cronograma com profissionais

## 🟡 Fase 2 — Compras

### Eletrodomésticos
- [ ] Máquina lava e seca Samsung branca
- [ ] Geladeira inverse branca
- [ ] Cooktop
- [ ] Forno elétrico (opcional)
- [ ] Depurador de ar
- [ ] Micro-ondas

### Móveis
- [ ] Sofá retrátil preto 180cm
- [ ] Painel de TV (madeira + preto)
- [ ] Cama casal + colchão
- [ ] Cama solteiro + colchão
- [ ] Criados-mudos
- [ ] Escrivaninha quarto solteiro
- [ ] Mesa bistrô + cadeiras (varanda)
- [ ] Mesa de centro (opcional)

### Espelhos 🪞
- [ ] Espelho grande sala (120x80cm, moldura preta)
- [ ] Espelho banheiro (largura total da bancada)
- [ ] Guarda-roupa casal com portas espelhadas
- [ ] Guarda-roupa solteiro com porta espelhada

### Marcenaria
- [ ] Armários superiores e inferiores cozinha
- [ ] Armário superior lavanderia
- [ ] Guarda-roupa planejado casal (com espelho)
- [ ] Guarda-roupa planejado solteiro (com espelho)
- [ ] Nichos lavanderia

### Iluminação
- [ ] Spots LED 3000K (15 unidades)
- [ ] Fita LED 3000K (~20m)
- [ ] Pendente sala/jantar
- [ ] Luminárias quartos
- [ ] Arandela varanda

### Revestimentos
- [ ] Piso porcelanato/vinílico (~44m²)
- [ ] Porcelanato box banheiro
- [ ] Painel ripado (parede destaque)
- [ ] Tinta branca/off-white (galões)

### Banheiro
- [ ] Vaso sanitário
- [ ] Cuba
- [ ] Torneira preta fosca
- [ ] Chuveiro
- [ ] Box vidro temperado (perfil preto)
- [ ] Acessórios pretos (porta-toalha, etc.)

### Cozinha
- [ ] Bancada granito preto (~2m)
- [ ] Cuba inox embutir
- [ ] Torneira preta fosca
- [ ] Tanque/cuba lavanderia

### Decoração
- [ ] Tapete cinza claro (150x200)
- [ ] Cortinas voil branco (sala + quartos)
- [ ] Roupa de cama neutra
- [ ] Plantas/vasos (varanda)

## 🟢 Fase 3 — Execução da Obra

- [ ] Pontos elétricos e iluminação
- [ ] Gesso/sanca/forro
- [ ] Assentamento do piso
- [ ] Revestimento box banheiro
- [ ] Pintura geral
- [ ] Instalação painel ripado
- [ ] Instalação marcenaria cozinha
- [ ] Instalação guarda-roupas
- [ ] Instalação painel TV
- [ ] Louças e metais banheiro
- [ ] Box de vidro
- [ ] Espelhos (sala + banheiro)
- [ ] Instalação spots + fita LED + pendentes
- [ ] Bancada granito cozinha

## 🔵 Fase 4 — Finalização

- [ ] Entrega e montagem dos móveis
- [ ] Instalação de cortinas
- [ ] Colocação do tapete
- [ ] Posicionamento da decoração
- [ ] Limpeza pós-obra profissional
- [ ] Inspeção final (elétrica, hidráulica, acabamentos)
- [ ] Fotos do antes/depois
- [ ] Mudança! 🎉
